# 01 新生入学

> 维护人：小航项目组
更新日期：2026\-07\-14
数据来源：郑州航院官网 / 2025级新生入学手册 / 在校学生经验整理
说明：本文档为新生入学相关信息汇总，硬信息来自学校官网，经验内容整理自学长学姐分享。如有出入以官方通知为准。

## 报到须知

### 报到时间

- 本科新生：9月7日—8日报到注册 ⚠ 以官方为准

- 专升本新生：9月7日—8日报到注册 ⚠ 以官方为准

- 研究生新生：9月7日—8日报到注册 ⚠ 以官方为准

- 第二学士学位：9月5日—6日报到注册 ⚠ 以官方为准

**注意事项：**

- 因故不能按期报到者，须提前向招生办请假，假期不得超过两周

- 未请假或请假逾期未报到者，除不可抗力等正当事由外，视为放弃入学资格

- 建议按通知书指定日期当天报到，不要提前来校，提前到校无法安排住宿

### 报到地点

- 龙子湖校区（主校区）：河南省郑州市郑东新区文苑西路15号 ⚠ 以官方为准

- 大学路校区（南校区）：河南省郑州市大学中路2号 ⚠ 以官方为准

- 具体报到场馆以学院通知为准，通常在体育馆或图书馆前广场

### 报到必备材料

1. 录取通知书（原件，务必带）

2. 高考准考证（原件备查）

3. 身份证及复印件（建议提前复印3份）

4. 一寸、二寸免冠证件照（蓝底/红底各1版，电子版也存好）

5. 党团组织关系转移材料（党员/团员需要）

6. 户口迁移证（自愿迁户口的同学带）

7. 档案（部分省份由高中统一邮寄，自带的务必完好移交）

8. 贫困证明/助学贷款材料（申请助学金、助学贷款的同学）

### 报到流程

**线上预报到（开学前完成）：**

1. 登录智慧迎新系统，填写个人信息

2. 上传证件照，完成线上信息核验

3. 选择宿舍床位（部分学院支持线上选房）

4. 完成线上缴费（学费、住宿费等）

5. 查看分班、分宿舍结果

**现场报到（到校当天）：**

1. 找到所在学院的迎新摊位，核验身份

2. 领取校园一卡通、宿舍钥匙、军训服领取券

3. 办理宿舍入住，到宿舍楼管处登记

4. 领取军训服装（按尺码试穿，不合适当场换）

5. 班会/年级大会（通常报到当天晚上开）

**学长经验：**

- 报到当天人很多，建议上午早点到，避开中午高峰

- 家长可以进校，但宿舍可能不让家长进，提前有个心理准备

- 校园里有志愿者学长学姐带路，找不到就问，别不好意思

- 当天会有很多推销的（电话卡、英语报、驾校等），别急着买，先了解清楚

## 宿舍安排

### 校区分布

- 龙子湖校区：大部分学院在东校区，设施较新

- 大学路校区：部分学院在南校区，位置靠近市区

### 宿舍条件

- 宿舍类型：以4\-6人间为主，具体看楼栋

- 床铺规格：0\.9m×2m，上下铺或上床下桌，因楼而异

- 基础设施：书桌、衣柜、空调、公共卫生间/独立卫浴因楼而异

- 热水供应：宿舍楼有公共热水房，部分楼层有淋浴间

- 用电限制：有功率限制，大功率电器（吹风机、电热锅等）会跳闸，严禁使用

### 宿舍费用

- 住宿费：约800\-1200元/年，按宿舍类型收取 ⚠ 以官方为准

- 电费：需自行充值，宿舍人均摊

- 水费：部分楼栋免冷水费，热水单独计费

### 宿舍相关注意事项

1. 宿舍分配通常按学院、班级集中安排，同班同学尽量住一起

2. 想换宿舍的，开学后申请，走宿舍调整流程，刚开学不让随便换

3. 宿舍门禁：通常晚上11点关门，晚归要登记

4. 查寝：学院会不定期查寝，夜不归宿会被通报

5. 宿舍里的空调遥控器、钥匙弄丢了要赔钱，保管好

**学长避坑：**

- 别信开学前网上"提前选宿舍"的谣言，一切以学校官方系统为准

- 被褥可以自带也可以到校买，学校卖的质量一般但方便，家远的建议到校买

- 宿舍楼下自动贩卖机的东西比超市贵，买得多去校园超市

- 第一周宿舍网络可能没装好，提前备点流量

## 学费缴纳

### 缴费项目

- 学费：普通专业约5000\-5700元/年，艺术类等专业更高 ⚠ 以官方为准

- 住宿费：800\-1200元/年，按实际住宿标准收取 ⚠ 以官方为准

- 体检费：约几十元，入学体检用 ⚠ 以官方为准

- 其他：教材费等据实收取，部分专业有实验材料费

### 缴费方式

**官方渠道（唯一推荐）：**

1. 微信关注"郑州航空工业管理学院财务处"公众号

2. 点击底部菜单"学生缴费"

3. 新生用用户名登录，在校生用学号登录

4. 选择缴费项目，微信支付完成

**注意：**

- 缴费时间：录取通知书发放后即可开始缴费，建议开学前缴清

- 发票：缴费后可在系统里申请电子发票

- 缓缴：家庭困难的同学可以走绿色通道，先入学后缴费，申请助学贷款

### 助学贷款

- 国家助学贷款分为生源地贷款和校园地贷款，建议办生源地的更方便

- 贷款额度：本专科生每年最高不超过12000元

- 办理流程：暑假期间在户籍所在地教育局/资助中心办理

- 到校后凭贷款受理证明走绿色通道报到，不用先交学费

**防骗提醒：**

- 任何要求你转账到"私人账户"交学费的都是诈骗

- 辅导员不会私下找你要学费，所有缴费走官方系统

- 收到"学费减免""退费"的电话短信，先打官方电话核实

## 军训须知

### 军训时间

- 本科新生：9月10日—9月30日，约21天 ⚠ 以官方为准

- 具体安排以军训团通知为准，遇恶劣天气会调整

### 军训地点

- 学校内操场、篮球场等场地进行，不需要去部队

- 按学院编成连队，统一训练

### 军训服装

- 包含：迷彩服外套、裤子、T恤、帽子、腰带、胶鞋

- 报到时凭券领取，现场试穿尺码，不合适当场调换

- 建议备一双鞋垫，军训鞋比较硬，站一天脚疼

### 军训内容

- 队列训练：站军姿、稍息立正、停止间转法、齐步正步跑步

- 内务整理：叠被子、宿舍卫生检查

- 军事理论学习、安全教育

- 最后有阅兵/会操表演

### 军训注意事项

1. 防晒！防晒！防晒！郑州9月太阳还是很毒，男生也别硬扛

2. 多喝水，训练场有供水点，带个大水杯

3. 身体不舒服立刻打报告休息，别硬撑，中暑了得不偿失

4. 军训算学分，无故缺勤会影响成绩，真有病凭医院证明请假

5. 头发要求：男生短发，女生不能披散头发，要扎起来塞帽子里

**学长经验：**

- 鞋垫买厚的，或者用卫生巾垫，亲测有效

- 防晒霜买SPF50\+的，每两小时补一次，不然会晒伤

- 带点润喉糖，喊口号喊多了嗓子疼

- 军训期间统一作息，趁这个机会调整生物钟，别熬夜

### 电子数码类

- 手机、充电器、数据线

- 郑州冬天有暖气，室内不冷，羽绒服不用急着带

- 很多东西学校超市都能买到，别从家搬一堆过来

## 新生常见问题

**Q：可以提前到校吗？**
A：不建议提前来，提前到校宿舍没开放没法入住。按通知书上的日期当天到就行。

**Q：家长能进宿舍吗？**
A：报到当天部分楼栋允许家长进，之后一般不让男生进女生宿舍、反之亦然。

**Q：校园卡丢了怎么办？**
A：先在一卡通系统挂失，然后带身份证到信息中心办事大厅补办，补卡要工本费。

**Q：转专业难吗？**
A：大一结束有一次转专业机会，要看大一成绩，绩点够高才有资格，具体看教务处政策。

**Q：学校有几个食堂？好吃吗？**
A：龙子湖校区有多个食堂，各有特色，价格适中，周边也有小吃街，不用担心吃不好。

**Q：宿舍能装宽带吗？**
A：可以，开学后运营商会在学校摆摊办宽带，自己选合适的套餐。

**Q：开学要带多少现金？**
A：不用带太多现金，学校基本都能手机支付。备个两三百应急就行。
